● Артикул создан

1 · Выбор карточек

Введите артикул и нажмите Enter (или кнопку справа) — загрузятся подходящие карточки. Оставьте пусто, чтобы загрузить все.

2 · Что меняем

1 · Категория и общие поля

2 · Создание из Честного ЗНАка (автоматически)

Загрузите Excel-файл из ЧЗ (после присвоения ГТИНов) — по каждому артикулу автоматически подтянутся размеры, баркоды (ГТИН), состав, цвет, ТНВЭД и наименование. Состав, цвет и ТНВЭД сами проставятся в характеристики карточки.

Тянет твои товары с присвоенными ГТИНами автоматически (нужен ключ ЧЗ в настройках). Файл загружать не нужно.

или загрузить файл вручную (.xlsx)

Перед созданием выберите категорию WB в блоке 1.

3 · Доп. характеристики (необязательно)

Состав, цвет и ТНВЭД подтягиваются из файла автоматически. Здесь можно добавить недостающие обязательные характеристики категории (применятся ко всем карточкам).

4 · Создать в разных цветах на основе действующей карточки

Найдите готовую карточку-образец (например, действующую худи) — её характеристики, габариты и размеры скопируются. Укажите цвета: для каждого создастся отдельная карточка с автогенерацией артикула JOTO и новыми баркодами.

Характеристики образца

Скопированы из найденной карточки — можно поправить. Поле «Цвет» подставится автоматически для каждого цвета.

Как пользоваться

Раздел «Карточки WB» — всё про товары на Wildberries в одном месте.

🅙
Генерация артикулов
Выберите категорию и цвет → «Создать артикул» — получите следующий свободный артикул JOTO. Можно добавить свою категорию (➕ Новая категория) и удалить её (🗑).
✎
Редактирование карточек
«Загрузить карточки с WB» → отметьте нужные галочками → в списке «Что меняем» выберите поле (SEO-описание, название, артикул, категория, характеристика, цена) → введите значение → «Проверить и отправить» → посмотрите предпросмотр → «Подтвердить». Изменения уходят на Wildberries.
＋
Создание карточек
Выберите категорию WB и общие поля → загрузите файл из Честного ЗНАка (после присвоения ГТИНов) → проверьте предпросмотр (артикул, размеры, баркоды, состав, цвет, ТНВЭД подтянутся сами) → «Создать карточки из файла на WB».
✓
Чек-лист
Готовность карточек по артикулам: фото, описание, характеристики, баркод, СЕО и т.д. Зелёный квадрат — заполнено, красный — нет. Можно фильтровать и отмечать пункты вручную.
⚠️ Для редактирования и создания карточек токен Wildberries (в настройках) должен иметь доступ к категории «Контент» (без галочки «Только на чтение»). Для смены цены — ещё и «Цены и скидки».
