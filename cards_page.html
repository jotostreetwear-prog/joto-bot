3 · Баркоды

Баркоды можно сгенерировать в Wildberries (кнопка в таблице ниже), получить ГТИНы из Честного ЗНАка по API, либо загрузить файл из Честного ЗНАка — тогда таблица заполнится автоматически (артикул + ГТИН).

Берём артикул («Модель / артикул производителя»), размер и присвоенный ГТИН (из «Кода товара» или «Результата обработки»). Одна карточка = один артикул со всеми размерами. Грузите файл уже после присвоения ГТИНов в ЧЗ.
