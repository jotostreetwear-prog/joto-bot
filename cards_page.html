Генерация артикула

Выберите категорию и цвет — система выдаст следующий свободный артикул JOTO.

● Артикул создан

Создали по ошибке? «Удалить артикул» освободит этот номер — он выдастся снова. Освободить можно только последний созданный номер.

Если менеджер создал артикул по ошибке — впишите номер, и он освободится (выдастся снова). Сработает только для последнего созданного номера в категории.

1 · Выбор карточек

Введите артикул и нажмите Enter (или кнопку справа) — загрузятся подходящие карточки. Оставьте пусто, чтобы загрузить все.
Выбрано: 0

2 · Что меняем

Шаг 1 · Карточка-образец

Найдите действующую похожую карточку — из неё скопируются категория, бренд, габариты, описание и характеристики. Из файла ЧЗ (шаг 2) подставятся артикулы, ГТИНы (баркоды) и размеры. Порядок: Марина генерит артикулы → Даша заводит ЧЗ → менеджер грузит файл → создаём.

Характеристики из образца

Применятся ко всем создаваемым карточкам — можно поправить. Цвет, состав и ТНВЭД по каждому артикулу возьмутся из файла ЧЗ (перекроют эти значения).

Шаг 2 · Общие поля и файл из Честного ЗНАка

Менеджер загружает Excel-файл из Честного ЗНака (с уже присвоенными ГТИНами) — по каждому артикулу подтянутся размеры, баркоды (ГТИН), состав, цвет, ТНВЭД и наименование. Состав, цвет и ТНВЭД сами проставятся в характеристики карточки. Баркодами становятся ГТИНы из файла.

Перед созданием заполните категорию WB и общие поля выше.

Доп. характеристики (необязательно)

Состав, цвет и ТНВЭД подтягиваются из файла автоматически. Здесь можно добавить недостающие обязательные характеристики категории (применятся ко всем карточкам).

Какие артикулы рекомендовать

По каждому артикулу: 🔗 Сейчас рекомендуется в товарах — в комплекте каких ваших товаров этот артикул уже стоит, 🧩 Что советует бот — комплект (дополняющие товары для рекомендаций), и ✅ Статус — ручная отметка менеджера. Выберите артикул или категорию из списка.

Wildberries закрыл API рекомендаций, поэтому бот их сам не проставляет. Порядок: бот советует комплект → менеджер вносит рекомендации вручную в кабинете WB → отмечает здесь статусом (новое → согласовано → поставлено на WB) для контроля.

Массовые действия на WB (рекомендации «Магазин рекомендует»)

«Добавить» — допишет артикул в рекомендации (не стирая прочее). «Очистить» — удалит все рекомендации у выбранных карточек. Действия применяются сразу на Wildberries.

Чек-лист запуска новинки на Wildberries

По каждому артикулу-новинке — все этапы запуска по регламенту с галочками. Отметки общие для всех менеджеров и сохраняются (видно, кто и когда отметил).

Как пользоваться

Раздел «Карточки WB» — всё про товары на Wildberries в одном месте.

🅙
Генерация артикулов
Выберите категорию и цвет → «Создать артикул» — получите следующий свободный артикул JOTO. Можно добавить свою категорию (➕ Новая категория) и удалить её (🗑).
✎
Редактирование карточек
«Загрузить карточки с WB» → отметьте нужные галочками → в списке «Что меняем» выберите поле (SEO-описание, название, артикул, категория, характеристика, цена) → введите значение → «Проверить и отправить» → посмотрите предпросмотр → «Подтвердить». Изменения уходят на Wildberries.
＋
Создание карточек
Выберите категорию WB и общие поля → загрузите файл из Честного ЗНАка (после присвоения ГТИНов) → проверьте предпросмотр (артикул, размеры, баркоды, состав, цвет, ТНВЭД подтянутся сами) → «Создать карточки из файла на WB».
✓
Чек-лист
Готовность карточек по артикулам: фото, описание, характеристики, баркод, СЕО и т.д. Зелёный квадрат — заполнено, красный — нет. Можно фильтровать и отмечать пункты вручную.
⚠️ Для редактирования и создания карточек токен Wildberries (в настройках) должен иметь доступ к категории «Контент» (без галочки «Только на чтение»). Для смены цены — ещё и «Цены и скидки».
